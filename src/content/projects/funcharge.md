## Overview

FunCharge makes it practical and fun for EV owners to plan charging stops on longer routes. Instead of just finding the nearest charger, it surfaces the full picture: efficiency of route, ranked charging options, and nearby activities to do while waiting.

## Features

- Optimal route calculation with charging stop recommendations
- Charging station ranking by availability, speed, and proximity
- Nearby activity suggestions during charge time
- Emergency contact integration for safety on long routes
- Responsive design for mobile use on the road

## Tech

Built with a Node.js backend and React frontend, integrating mapping APIs for routing and geolocation, and MongoDB for storing station data and user preferences.
